SCLS Graphic Benoit

SCLS Graphic Benoit is the graphic library of the SCLS project. It is made by Aster System, a little french organisation. As every SCLS project, it is made to be simple to use and understand. However, let's be honest, some graphics features are quite hard to understand. In this document, we will explain how the library works, and how to use it.

Core features

Graphism with OpenGL

The core of the library is to provide a simple way to use some graphic features with C++, with Windows and Linux operating system. For that, SCLS Graphic Benoit uses the OpenGL graphic library. OpenGL is a quite complex library, and it's not possible to explain it in a little paragraph. This page explains more how SCLS Graphic Benoit uses OpenGL.

GUI support

To be honest, use the core features of SCLS Graphic Benoit without anything is completely useless. However, SCLS Graphic Benoit offers a built-in GUI system, made to be simple, but powerful. The GUI system uses a transform system, which can be used lightly and simply, or entirely and with a more sophisticated ways. This system is very complex to use in its whole, this page explains it in details.
