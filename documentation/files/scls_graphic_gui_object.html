SCLS Graphic GUI Object

Presentation

SCLS Graphic GUI Object allows the user to use built-in GUI objects.

Content

SCLS Graphic Object is axed around the class "GUI_Object" heritating from Object. It's the only notable thing in this file. However, this class is full of usefull things to handle GUI objects in the window.

HUD Object main functions

As in Object, the HUD_Object class as some must-see functions.

Functions

HUD_Object(Window* window_struct)
Most basic constructor of an HUD_Object, used to clone some objects if necessary. It is contained into the Window "window_struct".
See the Object documentation for more informations.
HUD_Object(HUD_Object* window_struct, std::string a_name, std::string texture_name, std::string vao_name)
Simple HUD_Object constructor, usable by the developer. The name of the HUD object is "a_name". It is contained into the Window "window_struct".
The Texture applied is the main texture "texture_name", and the VAO applied is the main VAO "vao_name".
See the Object documentation for more informations.
~HUD_Object()
Most basic destructor of an HUD_Object.

Code specification

This file define the macro "SCLS_GRAPHIC_GUI_OBJECT".
