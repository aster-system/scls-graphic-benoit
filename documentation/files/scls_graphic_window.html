SCLS Graphic Window

Presentation

SCLS Graphic Window contains the main class which allows to use a window with SCLS. It is made to be very easily usable, by anybody.

Content

SCLS Graphic Window is axed around the class "Window". It's the only notable thing in this file. However, this class is full of usefull things to do a window in only some lines.

Window main functions

Window has some must-see features, made to be simple but very usefull.
To display Objects, the class uses the system of page. A page is the first object rendered, which should have a list of childrens to display.

Functions

Window(int window_width, int window_height, std::string exec_path)
The most basic window constructore, which should be easily creatable for the user.
std::string current_page_name()
Return the name of the currently displayed page.
void set_current_page(std::string new_current_page)
Set the current page to "new_current_page".
