De l'énergie via les atomes

------------------------------------------------------ Dans cette partie, l'idée va être d'expliquer le plus simplement possible
comment de simples atomes peuvent produire une tel quantité d'énergie. ------------------------------------------------------ Si le noyau de l'atome est trop lourd, alors il devient instable. Certains
noyaux sont tellement lourds qu'ils peuvent se casser en deux si ils
rentrent en collisions avec un neutron. Ce phénomène est appelé "la
fission nucléaire induite". Ces noyaux sont appelés "noyaux fissiles". ------------------------------------------------------ Le seul élement fissile trouvable dans la nature est l'uranium 235.
