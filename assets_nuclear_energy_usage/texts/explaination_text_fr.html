De l'énergie via les atomes

------------------------------------------------------ Dans cette partie, l'idée va être d'expliquer le plus simplement possible
comment de simples atomes peuvent produire une tel quantité d'énergie. ------------------------------------------------------ Premièrement, il est important de se rappeler que toute la matière qui nous
entoure est constituée d'atomes. Ces atomes sont eux aussi constitués d'une
partie centrale nommée "le noyau" de l'atome. Ce noyau est lui même
constitué de particules composites, nommées les protons et les neutrons. ------------------------------------------------------ Certains noyaux sont stables dans le temps, tandis que d'autres, trop lourds
ou déséquilibrés, sont instables dans le temps. Les noyaux instables sont
radioactifs. Pour redevenir stable, il subissent ce qu'on appelle une
désintégration, éjectant diverses particules, ainsi que des ondes,
qui représentent la fameuse radioactivité.
