La base de la matière

------------------------------------------------------ Dans cette partie, l'idée va être d'expliquer le plus simplement possible
que ce qui compose la matière, et d'où peut provenir l'énergie. ------------------------------------------------------ Premièrement, il est important de se rappeler que toute la matière qui nous
entoure est constituée d'atomes. Ces atomes sont eux aussi constitués d'une
partie centrale nommée "le noyau" de l'atome. Ce noyau est lui même
constitué de particules composites, nommées les protons et les neutrons. ------------------------------------------------------ L'élement auquel appartient un noyau est défini par son nombre de protons.
Selon le nombre de neutron dans le noyau, l'atome en question est un isotope,
nommé selon le nombre de neutron dans le noyau. ------------------------------------------------------ Bien que les protons aient une charge positive, ils ne se repoussent pas.
En effet, les protons, ainsi que les neutrons, sont constitués de
particules plus petites nommées les quarks. Ces particules sont liées entre
elles via une force nommée la "force forte". Cette force permet la liaison
entre tous les quarks, et donc les protons et les neutrons.
